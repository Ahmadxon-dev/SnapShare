# 🚀 Snapshare

## MERN Stack App

Snapshare is a modern social media application.  
Users can create and interact with posts, connect with others, and manage their profiles in a clean and responsive environment.

---

## 📦 Tech Stack

### 🎨 Frontend
- React  
- Redux
- Tailwind CSS  

### ⚙️ Backend
- Node.js  
- Express  
- MongoDB  
- JWT  
- Cloudinary  

---

## 🧠 Features

- 🔐 **User Actions**  
  - Follow and unfollow users  
  - Search for users  
  - Update profile (name & profile image)  

- 📝 **Post Actions**  
  - Create, edit, and delete posts  
  - Like and comment on posts
